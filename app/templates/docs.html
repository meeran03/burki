Burki Voice AI API

Enterprise-grade Voice AI Infrastructure API. Build intelligent voice applications with powerful assistants and comprehensive call management.

API v1.0 - Stable Base URL: https://api.burki.com

Quick Start

Get started with the Burki Voice AI API in minutes. Create your first assistant and start handling voice calls.

1. Get API Key

Generate your API key from the dashboard to authenticate requests.

2. Create Assistant

Configure your AI assistant with custom prompts and LLM providers.

3. Handle Calls

Monitor and manage voice calls with real-time transcripts.

4. Analyze Data

Export call data and analyze performance metrics.

Authentication

The Burki API uses API key authentication. Include your API key in the Authorization header as a Bearer token.

curl https://api.burki.com/api/v1/assistants \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
Important

Keep your API keys secure. Do not share them publicly or include them in client-side code.

Rate Limits

API requests are rate limited to ensure service quality for all users.

1,000
Requests per hour
100
Concurrent calls
10MB
Max request size

Rate limit headers are included in all responses:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200

Error Handling

The API uses conventional HTTP response codes to indicate success or failure.

200 OK - Request successful
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
404 Not Found - Resource not found
429 Too Many Requests - Rate limit exceeded

Error Response Format

{
  "detail": "Assistant with ID 123 not found in your organization",
  "error_code": "ASSISTANT_NOT_FOUND",
  "timestamp": "2025-01-03T10:30:00Z"
}

POST

Create Assistant

Create a new AI assistant with custom configuration and LLM provider settings.

curl https://api.burki.com/api/v1/assistants \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sales Assistant",
    "system_prompt": "You are a professional sales assistant...",
    "llm_provider": "openai",
    "llm_provider_config": {
      "model": "gpt-4",
      "temperature": 0.8
    },
    "voice_id": "alloy"
  }'

Request Body

Field Type Required Description
name string Yes Assistant name
system_prompt string Yes System prompt for the assistant
llm_provider string Yes LLM provider (openai, anthropic, gemini, etc.)
llm_provider_config object Optional Provider-specific configuration
voice_id string Optional Voice ID for TTS

PUT

Update Assistant

Update an existing assistant's configuration. Only the fields provided will be updated.

curl -X PUT https://api.burki.com/api/v1/assistants/123 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Support Assistant",
    "system_prompt": "You are an enhanced customer support agent...",
    "llm_provider_config": {
      "temperature": 0.8
    },
    "is_active": true
  }'

Update Status Only

PATCH Quick status update endpoint 
curl -X PATCH "https://api.burki.com/api/v1/assistants/123/status?is_active=false" \
  -H "Authorization: Bearer YOUR_API_KEY"

Calls API

GET

List Calls

Retrieve calls with comprehensive filtering options including date ranges, duration, and customer phone numbers.

curl https://api.burki.com/api/v1/calls \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -G \
  -d "status=completed" \
  -d "date_from=2025-01-01T00:00:00Z" \
  -d "limit=50"

Query Parameters

Parameter Type Description
status string Filter by call status (ongoing, completed, failed)
assistant_id integer Filter by assistant ID
date_from datetime Filter calls from this date (ISO format)
date_to datetime Filter calls to this date (ISO format)
customer_phone string Filter by customer phone number
min_duration integer Minimum call duration in seconds
max_duration integer Maximum call duration in seconds

Python SDK Example

Here's a complete example of using the Burki API with Python:

import requests
import json

class BurkiAPI:
    def __init__(self, api_key, base_url="https://api.burki.com"):
        self.api_key = api_key
        self.base_url = base_url
        self.headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
    
    def create_assistant(self, name, system_prompt, llm_provider="openai"):
        data = {
            "name": name,
            "system_prompt": system_prompt,
            "llm_provider": llm_provider,
            "llm_provider_config": {
                "model": "gpt-4",
                "temperature": 0.7
            }
        }
        response = requests.post(
            f"{self.base_url}/api/v1/assistants",
            headers=self.headers,
            json=data
        )
        return response.json()
    
    def get_calls(self, **filters):
        response = requests.get(
            f"{self.base_url}/api/v1/calls",
            headers=self.headers,
            params=filters
        )
        return response.json()
    
    def get_analytics(self, period="7d"):
        response = requests.get(
            f"{self.base_url}/api/v1/calls/analytics",
            headers=self.headers,
            params={"period": period}
        )
        return response.json()

# Usage
api = BurkiAPI("your_api_key_here")

# Create an assistant
assistant = api.create_assistant(
    name="Customer Support Bot",
    system_prompt="You are a helpful customer support assistant..."
)

# Get recent calls
calls = api.get_calls(status="completed", limit=10)

# Get analytics
analytics = api.get_analytics(period="30d")
print(f"Success rate: {analytics['summary']['success_rate']}%")

Call Transcripts

Get transcripts for a specific call with filtering options.

curl https://api.burki.com/api/v1/calls/123/transcripts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -G \
  -d "speaker=user" \
  -d "include_interim=false"

Query Parameters

Parameter Type Description
speaker string Filter by speaker (user/assistant)
include_interim boolean Include interim (non-final) transcripts

Export Transcripts

Text Format
curl https://api.burki.com/api/v1/calls/123/transcripts/export?format=txt \
  -H "Authorization: Bearer YOUR_API_KEY" \

Search Calls

Search calls by various criteria including call SIDs, phone numbers, and assistant names.

curl https://api.burki.com/api/v1/calls/search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -G \
  -d "q=+1234567890" \
  -d "limit=20"

Query Parameters

Parameter Type Description
q string Search query (minimum 3 characters)
limit integer Maximum number of results (1-200)
Search Tips

Search across call SIDs, customer phone numbers, and assistant names. Use partial matches for flexible searching.
